gatelink web push

To send an notification use the api client and POST notifications or

curl -X POST "http://localhost:8080/notifications" -H "accept: */*" -H "Content-Type: text/plain" -d "hello"

After gatelink server restart, you will have to re-register with the "subscribed to push notifications" checkbox.

Supported browsers

webpush API is supported by chrome, firefox and edge. See: Push API support.

For apple / safari support you will have to use the Apple Push Notifications Service, checkout: https://developer.apple.com/notifications/safari-push-notifications/
